# Personal-Covid-Tracker

## Dependencies<br />
**For backend** <br/>

    - cors
    - dotenv
    - mongodb
    - express
    - nodemon

**For frontend**<br />
    
    - react-router-dom
    - material-ui/core
    - material-ui/icons
    - axios

## How to run this project <br />
**For backend** <br/>

```
    cd backend
    npm install
    nodemon server 
```

**For frontend**<br />

```
    cd frontend
    npm install
    npm start
```
